## 一句话说清楚

> 把 PPT 定制这个非标服务，拆解为**需求翻译 → 风格匹配 → 内容生成 → 质量控制**四个可标准化环节，每个环节由 AI 和规则引擎驱动，形成可复制的交付流程。

## 为什么值得看

这不是一个"AI 做 PPT"的 demo。这是一个**真实商业环境中跑出来的系统**：

- **规模**：100+ 付费订单，不是玩具项目
- **验证**：客户满意度 100%，退款率 0%
- **迭代**：22 个典型案例 → 3 类核心问题诊断 → 3 套标准化对策
- **沉淀**：提示词模板库、客户分类方法论、效率优化模型

## 核心模块

| 模块 | 解决的问题 | 产出 |
|------|-----------|------|
| [提示词工程](prompts/) | 客户原始文字 → 结构化内容大纲 | 百单验证的 prompt 模板 |
| [案例诊断](cases/) | 22 个真实翻车案例的分析与修复 | 3 类问题标准化对策 |
