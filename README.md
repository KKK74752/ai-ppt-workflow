# AI-PPT Workflow

> 用 AI 系统化改造 PPT 定制交付流程的实践记录。

## 概述

一个完整的 AI 辅助内容生产案例——从需求分析、提示词工程、风格匹配到质量控制和效率优化。

覆盖 100+ 真实商业订单的数据沉淀。

## 目录

- [提示词模板](prompts/) — 经过百单验证的结构化 prompt
- [案例库](cases/) — 22 个典型案例，含问题诊断和解决路径
- [效率分析](analysis/) — 基于真实数据的流程优化方法论

## 技术栈

- **AI**: 结构化 prompt engineering、风格特征提取、内容大纲生成
- **工具链**: Git 版本追踪、数据驱动的效率分析
- **方法论**: 将模糊需求转化为可执行方案的标准化流程
