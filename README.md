# AI-PPT Workflow

> 一名 PPT 设计师用 AI 系统化提效的真实工作记录。
> 不是教程，不是作品集——是证据链。

## 我是谁

柯少荣。21 岁。湖北交通职业技术学院计算机应用技术专业毕业。在南京南大尚诚做过一年数字乡村项目。PPT 设计不是新手——以前就做过，2026 年重新捡起来，在电商平台累计完成 100+ 单。

这个仓库记录的是我**用 AI 把 PPT 做成系统化流程**的过程。

## 这个仓库是什么

我把每次接单中**AI 真实参与的部分**记录在这里：

- **prompts/** — 在实际订单中反复打磨的提示词模板
- **cases/** — 真实客户对话 → 处理方式 → 结果
- **analysis/** — 从实际数据中提炼的效率规律

## 为什么做这个

面试官问我"你会用 AI 吗"，我可以给他看一份简历。也可以给他看这个仓库。

第二个选项更有说服力。
