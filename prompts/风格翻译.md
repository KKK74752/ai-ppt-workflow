# 风格翻译

## 问题

客户说"要好看"、"高端大气"、"简约一点"——这类模糊描述是 PPT 设计最大的时间杀手。

## 解决方案

不让客户自由描述。用 AI 生成 3 种差异化风格卡片，让客户做选择题。

## Prompt

```
我需要你为一个PPT项目生成3种视觉风格方案。客户信息如下：

主题：{客户填的主题}
行业：{客户填的行业/场景}
对象：{客户填的受众}

请生成3种风格，每种包含：
- 风格名称（4字以内）
- 主色方案（给出16进制色值）
- 排版倾向（图文比例、对齐方式）
- 适合场景（一句话）
- 不适合场景（一句话）

3种风格必须有明显差异。

只输出方案，不加评价，不加推荐。
```

## 效果

| 之前 | 之后 |
|------|------|
| 客户说"不是这种感觉"重做整版 | 客户三选一，锁定方向再开工 |
| 平均驳回 3 版 | 平均驳回 1 版 |

## 真实案例

- 客户：文艺范（竞选→作品集方向变更）
- 如果当时先出风格卡片让她选，就不会在发源文件后改方向
