# 风格翻译

## 解决的问题

客户说"要好看""高端大气""简约一点"——模糊审美描述是定制服务最大的时间杀手。

## 方案

不要求客户描述想要什么（他们往往说不清），而是生成 3 种差异化方案让客户选择。

## Prompt

```
我需要为一个PPT项目生成3种视觉风格方案。

主题：{客户填的主题}
行业：{客户填的行业}
对象：{客户填的受众}

请生成3种风格，每种包含：
- 风格名称（4字以内）
- 主色方案（16进制色值）
- 排版倾向（图文比例、对齐方式）
- 适合场景（一句话）
- 不适合场景（一句话）

3种风格必须有明显差异。
只输出方案，不评价不推荐。
```

## 为什么有效

| 传统方式 | 风格卡片方式 |
|---------|------------|
| "你想要什么风格？" → 客户说不清 | "这三个方向你喜欢哪个？" → 客户能判断 |
| 设计师猜 → 做一版 → 驳回 → 重做 | 客户选 → 锁定方向 → 开工 |
| 平均 2-3 轮对齐 | 1 轮锁定 |

## 核心洞察

**客户不擅长创作，但擅长判断。** 把"你想要什么"改成"这几个里哪个接近你的想法"，需求对齐效率提升显著。
