# 结构化大纲

## 适用场景

客户给一段长文字（述职报告、论文摘要、项目介绍），要求做成 PPT。

## Prompt

```
请你帮我完成PPT页面中文字结构逻辑的梳理，并总结提炼结论型小标题。

【结构梳理】
首先，根据文本信息，按照不同的信息主题，区分文段的结构。分为N个版块。
其次，将文本内容按信息关联性重组。
要求：
1. 无关信息单独分组
2. 用缩进体现层级关系
3. 务必保留原文信息，确保观众阅读时能理解文段信息，可适当删除连接词、形容词

【标题提炼】
为每个内容模块提炼结论型标题，要求：
1. 控制在12字以内
2. 包含关键数据或核心结论
3. 使用"动词+结果"结构（如：市场规模突破X亿、用户增长超预期）

【总标题】
基于所有模块核心结论，提炼一个18字内的总结性标题，要求：
1. 使用"结论先行+价值点"结构（如：用户留存率提升30%，实现精细化运营突破）
2. 内容精简但总体意思不变

【原始文本】
{客户提供的文字内容}
```
